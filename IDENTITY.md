# React 源码深度解析 - 项目身份

## 项目名称
**React Source Code Analysis** (React 源码深度解析)

## 项目定位

参考「React 技术揭秘」的实现方式，全面更新 React 18/19 新特性与源码架构的中文学习指南

## 目标受众

- 有一定 React 使用经验，想深入理解原理的开发者
- 想学习 React 18 Concurrent Features 的开发者
- 想跟进 React 19 新特性的开发者
- React 源码贡献者/学习者

## 核心价值

1. **最新版本**：覆盖 React 18 稳定版和 React 19 最新特性
2. **渐进式学习**：从理念 → 架构 → 实现，自顶向下
3. **可交互 Demo**：每个核心概念配有可运行的在线示例
4. **图解丰富**：大量架构图、流程图、时序图

## 内容特色

| 特色 | 说明 |
|------|------|
| 版本更新 | 覆盖 React 18/19，而非旧的 v17 |
| 架构图解 | 每个核心模块配有可视化的架构图 |
| 源码调试 | 手把手教你调试 React 源码 |
| 在线 Demo | 关键概念配有 CodeSandbox 可交互示例 |
| 中文友好 | 专为中文开发者编写，无理解障碍 |

## 与原版「React 技术揭秘」的区别

| 维度 | 原版 (v17) | 本版本 |
|------|-----------|--------|
| React 版本 | v17.0.0-alpha | v18.x + v19.x |
| Concurrent | 简单介绍 | 深度讲解 |
| Hooks | 基础 Hooks | 全部 Hooks + 新增 Hooks |
| Suspense | 基础 | Suspense + concurrent features |
| React 19 | ❌ | ✅ 完整覆盖 |
| 编译器 | ❌ | ✅ React Compiler 原理 |
